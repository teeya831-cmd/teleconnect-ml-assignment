# TeleConnect Customer Churn Prediction & Revenue Forecasting

## Project Overview

This project predicts customer churn and forecasts customer revenue using machine learning techniques. Multiple classification and regression models were evaluated, and model interpretability was performed using SHAP.

## Dataset

Telecom customer dataset containing customer demographics, services, contract information, billing details, and churn status.

## Techniques Used

* Data Cleaning
* Feature Engineering
* One-Hot Encoding
* Feature Scaling
* SMOTE for Class Balancing
* Classification Models
* Regression Models
* SHAP Interpretation
* Hyperparameter Tuning

## Best Models

### Churn Prediction

* Random Forest Classifier
* Accuracy: 85.12%
* ROC-AUC: 0.925

### Revenue Forecasting

* Random Forest Regressor
* R² Score: 0.998

## Business Recommendations

1. Retain short-tenure customers.
2. Encourage long-term contracts.
3. Monitor customers with high monthly charges.
4. Optimize service bundles.
5. Use revenue forecasting for strategic planning.
